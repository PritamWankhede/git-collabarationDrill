Git commands:
git init -> Initializes a new Git repository in the current directory.
git config -> Configures Git settings, such as user name and email.
git clone -> Clones a repository from a remote source to your local machine.
git add -> Adds files to the staging area, preparing them for a commit.
git commit -> Records the changes made to the files in the repository.
git status -> Shows the current status of the working directory and staging area.
git log -> Displays a log of all commits made to the repository.
git branch ->Lists all branches or creates a new branch.
git checkout ->Switches between branches
git merge -> Merges changes from one branch into another.
git push ->Pushes local commits to a remote repository.
git diff ->Shows the differences between commits.


CLI commands:
ls -> Lists the files and directories in the current directory.
cd -> Changes the current working directory.'
cp ->Copies files or directories from one location to another.
touch -> Creates a new empty file
mkdir -> Creates a new directory.
cat -> displays the content of files.
nano -> Opens the Nano text editor for file editing directly from the terminal.
chmod -> Changes file or directory permissions.
grep -> Searches for patterns within files or output.
ps -> Displays currently running processes.
find -> Searches for files and directories based on criteria like name
head -> display the first part of a file 
tail -> display the last part of a file 
pkill -> to terminate process
netstat -> provides information about current network activity and listening ports.
which -> locate the executable file


Computer Networks command:
ifconfig -> display network interface parameters, such as IP addresses and network settings
ping -> Tests connectivity between the local machine and a remote host.
hostname -> display hostname
ssh -> to securely connect to a remote machine over the SSH
host -> to perform DNS lookups, translating domain names into IP addresses
